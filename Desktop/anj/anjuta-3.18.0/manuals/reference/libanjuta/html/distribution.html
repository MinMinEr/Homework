Distribution

You can write your own project template and distribute them in a gzip compressed tar file. The archive has to included all files needed by your template.

With Anjuta version 2.30 or below, the user has to decompress this archive in a directory named anjuta/templates in the data directory defined according to XDG Base Directory Specification. With Anjuta version 2.31 or greater, if you name your archive with the extension .wiz.tgz the template will be automatically installed and then executed if used as anjuta argument.
